# Star Stable Linux 🐎

Unofficial, native Linux launcher for Star Stable Online.

## Requirements
- Python 3
- PyQT6
- Wine and winetricks

## Usage
- For every other distro: `python3 star-stable-linux-gui.py`
- For NixOS: `nix-shell`
